# brentcassity.com

**Personal website for Brent Cassity** - Author, Podcast Host, Speaker

**Live Site:** https://brentcassity.com

## About

Brent Cassity helps people break down the prisons in their own mind through his speaking series, book *Nightmare Success*, and podcast featuring stories of redemption and second chances.

- **Book:** [Nightmare Success on Amazon](https://amzn.to/3WC9gg3)
- **Podcast:** [nightmaresuccess.com](https://nightmaresuccess.com)
- **Speaking:** bdc@brentcassity.com

## Technology

- **Framework:** Astro 5.x
- **Styling:** Tailwind CSS 4.x
- **Hosting:** Cloudflare Pages
- **Deployment:** Push to GitHub → Auto-deploy

## Local Development

\`\`\`bash
npm install
npm run dev
\`\`\`

Open http://localhost:4321

## Build

\`\`\`bash
npm run build
\`\`\`

Outputs to \`/dist/\`

## Content Structure

- **Single-page site** with sections:
  - Hero (Breaking down the prisons in your own mind)
  - Meet Brent (biography)
  - Podcast (featured episodes + links to nightmaresuccess.com)
  - Speaking (testimonials + booking)
  - Book (Nightmare Success memoir)
  - Footer (social links + contact)

## SEO Features

- Schema.org markup (Person, Book, PodcastSeries)
- Open Graph tags
- Twitter Card tags
- Comprehensive llms.txt
- Sitemap + robots.txt
- Fast loading
- Mobile responsive
